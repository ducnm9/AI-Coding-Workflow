# Code Style — Kiro Steering

## Output Rules
- Code phải chạy được ngay, không cần chỉnh sửa
- Không giải thích dài dòng trừ khi được hỏi
- Khi fix lỗi: chỉ ra root cause trong 1-2 dòng
- Luôn dùng markdown code blocks với language tag
- Clean code, single responsibility
- Meaningful names, minimal comments
- Error handling đầy đủ
